Quick start

This page walks through a complete, realistic document from an empty script to a finished PDF. By the end you will have used the document, page, font, element and table APIs — which together cover most day-to-day work.

The shape of a script

Every Papier script follows the same four steps: create a document, register the fonts you intend to use, add pages and put elements on them, then save. Fonts are registered once at the document level and referred to afterwards by the name that addFont() returns.

require_once __DIR__ . '/vendor/autoload.php';

use Papier\PdfDocument;

$doc  = PdfDocument::create();          // 1. document
$font = $doc->addFont('Helvetica');     // 2. fonts
$page = $doc->addPage();                // 3. pages and content
$doc->save('out.pdf');                  // 4. save

Coordinates. PDF places the origin at the bottom-left of the page, with y increasing upwards, and measures everything in points (1 pt = 1/72 inch). A default A4 page is 595.28 × 841.89 pt, so at(72, 750) is one inch from the left edge and near the top.

Create the document

PdfDocument::create() returns an empty document targeting PDF 1.7. The metadata setters are fluent and write to the Info dictionary, which is what a viewer shows under "Document properties".

use Papier\PdfDocument;

$doc = PdfDocument::create();

$doc->setTitle('Invoice 2026-042')
    ->setAuthor('Acme Corporation')
    ->setSubject('Invoice for July 2026')
    ->setKeywords('invoice, billing, acme')
    ->setCreator('Acme Billing System');

Register fonts

Papier ships with the 14 standard Type 1 fonts, which every PDF viewer is required to provide. They need no embedding, so a document that uses only these stays small.

$regular = $doc->addFont('Helvetica');
$bold    = $doc->addFont('Helvetica-Bold');
$italic  = $doc->addFont('Helvetica-Oblique');

addFont() returns a resource name string — not an object — and that string is what you pass to an element's font() method. Registering the same font twice returns the same name, so there is no cost to calling it defensively.

To embed your own typeface, pass a path to a .ttf or .otf file instead. See Text & fonts for embedding, subsetting and Unicode.

Add a page

addPage() appends an A4 portrait page by default. Pass explicit dimensions in points for anything else — landscape A4 is simply the two arguments swapped.

$page = $doc->addPage();                    // A4 portrait
$wide = $doc->addPage(841.89, 595.28);      // A4 landscape
$us   = $doc->addPage(612, 792);            // US Letter

Draw the header

Content is composed from elements passed to $page->add(). It accepts any number of arguments and draws them in order, so later elements paint over earlier ones — put your background shapes first.

use Papier\Elements\{Color, Rectangle, Text};

$page->add(
    Rectangle::create(0, 780, 595.28, 62)->fill(Color::hex('#2c3e7a')),

    Text::write('ACME CORPORATION')
        ->at(40, 812)->font($bold, 20)->color(Color::white()),

    Text::write('123 Industrial Way · Paris · France')
        ->at(40, 793)->font($regular, 9)->color(Color::hex('#c9d2ee')),
);

Color builds colours four ways — rgb(), hex(), gray() and cmyk() — plus the black() and white() shorthands. Use cmyk() when the document is going to a print shop; the values are written to the file in the colour space you chose, without conversion.

Add body text

Text draws a single line at an exact position. TextBox takes a width and wraps the content across as many lines as it needs — use it for anything you did not author character by character.

use Papier\Elements\TextBox;

$page->add(
    Text::write('INVOICE')->at(40, 730)->font($bold, 28)->color(Color::hex('#1a1a2e')),
    Text::write('No. 2026-042')->at(40, 710)->font($regular, 11)->rgb(0.4, 0.4, 0.4),

    Text::write('Billed to')->at(40, 672)->font($bold, 10),
    TextBox::write("Jane Doe\nDoe Consulting SARL\n42 rue de la Paix\n75002 Paris")
        ->at(40, 656)->size(240)
        ->font($regular, 10)
        ->lineHeight(1.4),

    Text::write('Date')->at(400, 672)->font($bold, 10),
    Text::write('18 July 2026')->at(400, 656)->font($regular, 10),
);

TextBox::size() takes a width and an optional height. Leave the height at its default and the box grows downwards as far as the text requires; give it one and the text is clipped to that rectangle. lineHeight() is a multiplier of the font size, and align() accepts 'left', 'center' or 'right'.

Latin accented characters — é, à, ü, ñ — work without any configuration. For Greek, Cyrillic, Arabic or CJK you need a Unicode font; see Unicode and CJK.

Build a table

Table handles the line items. Table::create($x, $y) positions its top-left corner, setColumnWidths() defines the columns, and each call to addRow() appends a row of strings. Mark the first row as a header with setHeaderRows(1) and it gets the header font and background instead of the body styling.

use Papier\Elements\Table;

$table = Table::create(40, 600)
    ->setColumnWidths(255, 60, 90, 110)
    ->setFont($regular, 10, 'Helvetica')
    ->setHeaderFont($bold, 10, 'Helvetica-Bold')
    ->setHeaderRows(1)
    ->setHeaderBg(Color::hex('#2c3e7a'))
    ->setHeaderTextColor(Color::white())
    ->setAltRowBg(Color::hex('#f2f4fa'))
    ->setCellPaddingXY(8, 5);

$table->addRow(['Description', 'Qty', 'Unit price', 'Amount']);
$table->addRow(['Design system audit',      '1',  '2 400.00 €', '2 400.00 €']);
$table->addRow(['Component implementation', '12', '180.00 €',   '2 160.00 €']);
$table->addRow(['Documentation',            '4',  '180.00 €',   '720.00 €']);

$page->add($table);

Why the third argument to setFont()? The first is the resource name returned by addFont(); the third is the base font name Papier uses to look up character widths so it can measure text for wrapping and alignment. Pass the same name you gave addFont().

Individual cells can override the table defaults by passing a TableCell instead of a string. Tables also support footer rows, rowspan and colspan, per-side borders and opacity — see Tables.

Finish with graphics

Rules, boxes and shapes are elements like any other. Here a hairline separates the totals and a filled rectangle emphasises the amount due.

use Papier\Elements\Line;

$page->add(
    Line::from(40, 420)->to(555, 420)->color(Color::hex('#d0d4dd'))->width(0.75),

    Rectangle::create(355, 360, 200, 44)->fill(Color::hex('#f2f4fa')),
    Text::write('Total due')->at(370, 386)->font($regular, 11),
    Text::write('5 280.00 €')->at(370, 368)->font($bold, 16)->color(Color::hex('#2c3e7a')),

    Text::write('Payment due within 30 days.')
        ->at(40, 60)->font($italic, 9)->rgb(0.5, 0.5, 0.5),
);

Every shape element supports fill(), stroke() and opacity(); Line adds width() and dash(). For curves, clipping paths, gradients and blend modes, see Graphics.

Save or stream

Writing to disk is a single call:

$doc->save(__DIR__ . '/invoice.pdf');

To send the file to a browser instead, output() emits the headers and the bytes together. Pass false as the second argument to force a download rather than inline display:

$doc->output('invoice-2026-042.pdf');          // display in the browser
$doc->output('invoice-2026-042.pdf', false);   // download

And when you need the raw bytes — to return them from a framework controller, upload them, or feed them to a signer — use toString():

$bytes = $doc->toString();

The complete script

Everything above, in one file:

require_once __DIR__ . '/vendor/autoload.php';

use Papier\PdfDocument;
use Papier\Elements\{Color, Line, Rectangle, Table, Text, TextBox};

$doc = PdfDocument::create();
$doc->setTitle('Invoice 2026-042')
    ->setAuthor('Acme Corporation')
    ->setSubject('Invoice for July 2026');

$regular = $doc->addFont('Helvetica');
$bold    = $doc->addFont('Helvetica-Bold');
$italic  = $doc->addFont('Helvetica-Oblique');

$page = $doc->addPage();

$page->add(
    Rectangle::create(0, 780, 595.28, 62)->fill(Color::hex('#2c3e7a')),
    Text::write('ACME CORPORATION')->at(40, 812)->font($bold, 20)->color(Color::white()),

    Text::write('INVOICE')->at(40, 730)->font($bold, 28)->color(Color::hex('#1a1a2e')),
    Text::write('No. 2026-042')->at(40, 710)->font($regular, 11)->rgb(0.4, 0.4, 0.4),

    Text::write('Billed to')->at(40, 672)->font($bold, 10),
    TextBox::write("Jane Doe\nDoe Consulting SARL\n42 rue de la Paix\n75002 Paris")
